자바 스크립트의 데이터 타입의 확장 개념 집합데이터를 다룰 수 있는 객체

  • set 중복데이터 허용 x : 중복데이터가 들어오면 체크한 후에 하나마 남김
  • set() : 파라미터는 주로 배열 사용, 배열의 index,value 값을 활용하여 집합 데이터를 만듬
  • set 전용 메서드들이 제공
  • new Set() or new Set(Array) : set 객체를 생성 및 초기화
  • s.add(value) : 데이터 추가
  • s.delete(value) : 데이터 삭제
  • s.has(value) : 값의 존재 여부를 true/false 로 리턴
  • s.forEach(callback function) : set 순환
  • s.entries(), s.keys(), s.values() : 리턴타입 SetIterator => 별도 메서드로 접근하여 사용
  • s.size : set 사이즈
  • s.clear() : set 비우기
